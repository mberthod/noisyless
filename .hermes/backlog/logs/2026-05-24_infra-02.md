# infra-02 — MQTT TLS port 8883
**Timestamp** : 2026-05-24 21:41
**Statut** : ✅ Succès

## Contexte
Objectif : Configurer MQTT sur TLS (porte 8883) et générer une chaîne de confiance complète (CA → Serveur) sur le VPS `noisyless@91.99.26.43`.

## Actions réalisées

1. **SSH vers le VPS** : `ssh noisyless@91.99.26.43`
2. **Création du dossier certs** : `/opt/noisyless/mosquitto/certs/`
3. **Génération CA auto-signée** :
   - `openssl genrsa -out certs/ca.key 2048`
   - `openssl req -new -x509 -days 3650 -key certs/ca.key -out certs/ca.crt -subj "/CN=NOISYLESS-MQTT-CA"`
4. **Génération certificat serveur** :
   - `openssl genrsa -out certs/server.key 2048`
   - `openssl req -new -key certs/server.key -out certs/server.csr -subj "/CN=noisyless-mqtt"`
   - `openssl x509 -req -in certs/server.csr -CA certs/ca.crt -CAkey certs/ca.key -CAcreateserial -out certs/server.crt -days 365`
5. **Configuration mosquitto.conf** : Ajout du bloc TLS
   ```
   listener 8883
   cafile /etc/mosquitto/certs/ca.crt
   certfile /etc/mosquitto/certs/server.crt
   keyfile /etc/mosquitto/certs/server.key
   require_certificate false
   ```
6. **Redémarrage Mosquitto** : `docker compose restart mosquitto` → OK
7. **Vérification logs** : `docker compose logs mosquitto --tail 20` → service démarré sans erreur
8. **Test connexion SSL locale** : `mosquitto_pub -h localhost -p 8883 --cafile /opt/noisyless/mosquitto/certs/ca.crt -t "test/ssl" -m "hello" -d` → **SUCCÈS**
